# Explicación de la Práctica UT2 sobre los IDEs

En esta práctica tuvimos que hacer un trabajo individual sobre los Entornos de Desarrollo Integrados (IDE)

Un **IDE** es una aplicación de software que combina herramientas esenciales para programar en una interfaz gráfica.
El IDE nos facilita la creación de aplicaciones o programas al integrar un **editor de código, compilador, depurador y automatización de tareas**

En nuestro caso hemos utilizado a lo largo del curso:
- *Visual Studio Code* para **HTML, CSS, XML, DTD y XSD** para la asignatura de Lenguaje de Marcas.
- *NetBeans* para **Java** en la asignatura de programación.
