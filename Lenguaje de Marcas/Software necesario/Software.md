Software Necesario - Lenguaje de Marcas
Para el desarrollo práctico y seguimiento de la asignatura de Lenguaje de Marcas, utilizaremos exclusivamente la siguiente herramienta:

Visual Studio Code (VS Code)

Descripción: Es un editor de código fuente ligero pero muy potente, ideal para estructurar y editar documentos en HTML, XML, CSS y otros lenguajes de marcas.

Enlace de descarga oficial: https://code.visualstudio.com/

Guía básica de instalación
Descarga: Accede al enlace oficial proporcionado arriba y haz clic en el botón principal de descarga correspondiente a tu sistema operativo (Windows, macOS o Linux).

Ejecución: Una vez descargado, abre el archivo instalador (en Windows suele llamarse VSCodeUserSetup.exe) para iniciar el asistente.

Términos y condiciones: Lee y acepta el acuerdo de licencia para poder continuar.

Configuración de tareas adicionales: Durante la instalación, es muy recomendable marcar las casillas que dicen "Agregar la acción 'Abrir con Code' al menú contextual" del archivo y del directorio. Esto te permitirá abrir tus carpetas de apuntes directamente en el editor con un solo clic derecho.

Finalización: Haz clic en "Instalar", espera unos segundos a que se copien los archivos y pulsa "Finalizar" para lanzar el programa por primera vez.
